== Description ==

LTKG SEO Controller is a unified plugin that generates SEO-friendly long-tail keywords from your post content and title, and injects them into your preferred SEO plugin(s). Built with love by **Mona Galal**, this tool saves time and improves your on-page optimization.

**Supported SEO Plugins:**
- Yoast SEO (1 focus keyword)
- Rank Math (5 keywords)
- All in One SEO (5 keywords)
- WP Meta SEO (5 keywords + meta title + description)

**Features:**
- Automatic keyword generation on post save
- Manual keyword generation per post
- Bulk generation for all posts or today’s posts only
- Settings page to enable/disable plugins

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/ltkg-seo-controller`
2. Activate the plugin through the "Plugins" menu in WordPress
3. Go to "Settings > LTKG SEO Controller" to configure
4. Optional: Use the manual generate button in post editor or bulk generate from "Tools > LTKG Bulk SEO"

== الوصف ==

إضافة موحدة لتوليد الكلمات المفتاحية طويلة الذيل تلقائيًا من عنوان ومحتوى المقال، وتخزينها مباشرة في إضافات السيو المختلفة. مصممة لتوفير الوقت وتحسين السيو الداخلي للمحتوى.

== الإضافات المدعومة ==

- Yoast SEO (كلمة واحدة فقط)
- Rank Math (حتى 5 كلمات)
- All in One SEO (حتى 5 كلمات)
- WP Meta SEO (حتى 5 كلمات + عنوان ووصف ميتا تلقائي)

== الميزات ==

- توليد تلقائي عند حفظ المقال
- زر توليد يدوي في صفحة المقال
- توليد جماعي لجميع المقالات أو مقالات اليوم فقط
- إعدادات مرنة لاختيار الإضافات المستخدمة

== خطوات التثبيت ==

1. ارفع مجلد الإضافة إلى: `/wp-content/plugins/ltkg-seo-controller`
2. فعل الإضافة من لوحة "الإضافات"
3. اذهب إلى "الإعدادات > LTKG SEO Controller" لتخصيص الإعدادات
4. استخدم التوليد اليدوي من المقال، أو التوليد الجماعي من "أدوات > LTKG Bulk SEO"
